# Quiz Application

A simple Quiz Application built using Python and Tkinter.  
This project presents multiple-choice questions, tracks the score, and includes a timer for each question.

##  Overview

This application is a desktop-based quiz system developed to practice Python fundamentals and basic GUI development.

The quiz:
- Displays one question at a time
- Allows answer selection
- Shows feedback (correct / wrong)
- Applies a time limit per question
- Displays final score & percentage
- Supports restarting the quiz

Questions are loaded dynamically from a JSON file, making the system easy to extend.

## Features

✔ Graphical User Interface (Tkinter)  
✔ Multiple-choice questions  
✔ Score tracking  
✔ Timer per question  
✔ Answer feedback  
✔ Final result summary  
✔ Restart quiz option  
✔ JSON-based question bank  

##  Tech Stack

- **Language:** Python  
- **GUI Library:** Tkinter  
- **Data Storage:** JSON
